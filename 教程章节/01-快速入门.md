# Claude Code 快速入门

## 安装与配置

```bash
# 安装 Claude Code（VS Code 插件商店搜索安装）
# 或命令行安装
npm install -g @anthropic-ai/claude-code

# 初始化
claude init
```

## 基础操作

### 对话

直接在 Claude Code 面板输入问题或指令：
- `解释这段代码` — 代码理解
- `优化这个函数` — 代码优化
- `帮我写一个单元测试` — 测试生成

### 文件操作

```bash
# 创建文件
> 创建 src/utils.ts

# 编辑文件
> 在 login.ts 中添加表单验证

# 读取文件
> 读取 package.json 的依赖配置
```

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| Cmd+I | 打开 Claude Code 面板 |
| Cmd+Enter | 发送消息 |
| Cmd+K | 清除对话 |
| Cmd+[ | 撤销上一步操作 |

## Skill 系统

### 加载 Skill

```bash
# 列出可用 Skill
claude skill list

# 加载指定 Skill
claude skill load python-expert

# 创建自定义 Skill
claude skill create my-skill
```

### Skill 文件结构

```
my-skill.skill/
├── SKILL.md          # 核心定义
├── assets/           # 资源文件
└── templates/        # 模板
```

## 工作流示例

### 代码审查

1. 加载代码审查 Skill
2. 粘贴代码或指定文件
3. 自动分析并输出结果

### Bug 修复

1. 描述 Bug 现象
2. Claude 定位问题代码
3. 建议修复方案
4. 确认后自动应用修改

## 进阶技巧

- **上下文管理**：用 `#file:path` 引用文件
- **批量操作**：一次处理整个目录
- **自动化**：组合多个指令形成流水线
