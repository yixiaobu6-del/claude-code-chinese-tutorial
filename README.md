## Installation / 安装

无需安装，直接阅读 Markdown 教程文件。

```bash
# 克隆仓库
git clone https://github.com/yourusername/claude-code-chinese-tutorial.git

cd claude-code-chinese-tutorial

# 查看教程目录
ls 教程章节/
```
